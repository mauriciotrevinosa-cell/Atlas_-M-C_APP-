# Atlas Monte Carlo Simulation

Este repositorio contiene el módulo de simulación Monte Carlo de alto rendimiento.

## Descripción
Implementación avanzada de algoritmos de Monte Carlo para proyección financiera, incluyendo:
- Simulación de movimiento browniano geométrico (GBM)
- Modelos de salto Merton y volatilidad estocástica Heston
- Técnicas de reducción de varianza (Antithetic, Control Variates, Quasi-Random Sobol)

## Propósito
Demostrar habilidades en:
- Matemática Financiera Computacional
- Optimización de Rendimiento (NumPy/Pandas)
- Análisis Cuantitativo
