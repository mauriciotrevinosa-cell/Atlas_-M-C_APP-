# Atlas Aria (Basic Assistant)

Este repositorio contiene una versión modular del asistente de inteligencia artificial Aria, diseñado para el ecosistema Atlas.

## Descripción
Esta implementación demuestra cómo se estructura un agente inteligente capaz de interactuar con el usuario y otros módulos del sistema. Incluye la lógica de enrutamiento, herramientas básicas y manejo de comandos de voz/texto.

## Propósito
Demostrar habilidades en:
- Desarrollo de Agentes de IA
- Integración de LLMs y Herramientas
- Diseño de Interfaces Conversacionales
