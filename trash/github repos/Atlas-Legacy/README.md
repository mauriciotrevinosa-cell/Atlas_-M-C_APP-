# Atlas Legacy System

Este repositorio contiene la versión estable anterior (Legacy) del sistema Atlas.

## Descripción
Atlas Legacy representa la base sólida sobre la cual evolucionó el sistema actual. Incluye una arquitectura completa con módulos de UI, servicios y gestión de datos.

## Propósito
Este código se comparte para demostrar:
- Evolución y mantenimiento de software a largo plazo
- Capacidad para estructurar sistemas complejos desde cero
- Manejo de deuda técnica y refactorización

**Nota**: Este es un sistema archivado con fines demostrativos.
