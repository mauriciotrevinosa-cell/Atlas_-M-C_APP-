# Atlas Market Intelligence (Mini)

Este repositorio contiene el núcleo lógico de predicción de mercado y motores de probabilidad del proyecto Atlas.

## Descripción
Una versión compacta que extrae la lógica esencial para la generación de señales de trading y cálculo de probabilidades de mercado.
Incluye:
- `signal_engine.py`: Motor de detección y generación de señales técnicas.
- `prob_engine.py`: Motor de cálculo probabilístico de escenarios.

## Propósito
Demostrar habilidades en:
- Lógica de Trading Algorítmico
- Diseño de Sistemas de Inferencia
- Estructuración de Motores de Decisión
