# Secciones Del Proyecto

Resumen de las zonas principales de Atlas:

- `apps/desktop` - Interfaz principal para visualizar estado, modulos y herramientas.
- `apps/server` - API local para exponer estado del sistema y conectar datos.
- `python/src/atlas` - Motores, servicios, laboratorio y logica cuantitativa.
- `core/ai_assistant` - Agentes y prompts internos para apoyo de decisiones.
- `pixel-agents` - Componentes y utilidades relacionadas al ecosistema de Pixel Agents.
- `configs` - Configuracion operativa y referencias de entorno.
- `tests` - Validaciones unitarias e integracion.
- `project_governance` - Vision general de Atlas en modo flexible.
