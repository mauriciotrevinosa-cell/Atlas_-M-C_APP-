# Atlas Docs

Este folder guarda la documentacion viva de Atlas.

La idea es simple: notas claras, sin imponer reglas fijas, para que el proyecto pueda evolucionar rapido.

Secciones:
- `SECCIONES.md` - Mapa rapido de las partes del proyecto.
- `WORKFLOW_FLEXIBLE.md` - Forma de trabajo sugerida, moldeable.
- `ESTADO_ACTUAL.md` - Que ya esta conectado y que sigue.
- `IDEAS_ABIERTAS.md` - Ideas en curso para iterar.
